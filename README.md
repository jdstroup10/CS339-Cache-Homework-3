Build Instructions:

In command line terminal run the following commands:

Clone repository: ~ % git clone https://github.com/jdstroup10/CS339-Cache-Homework-3

Navigate to project folder: ~ % cd CS3339-Cache-Homework-3

Create the executable: ~ % g++ main.cpp -o main.out

Run the executable with three arguments- The number of entries, associativity, and the input file name:

Example: ./main.out 4 2 input1.txt
